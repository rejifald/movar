# Movar — довгий опис (українська)

Використовується у полі «Опис» на AMO, Chrome Web Store і Microsoft Edge Add-ons. EN- і UK-версії мусять збігатися за всіма фактичними твердженнями — список пошукових систем, мови, як працює, приватність. Див. [`../REQUIREMENTS.md`](../REQUIREMENTS.md) §2 щодо позиціювання та §4 щодо порядку розділів.

Status: first draft. Char count ≈ 1,624, well under AMO 15k / CWS 16k / Edge 10000.

---

Movar тримає інтернет тією мовою, якою ви насправді читаєте. Відкриваєте український сайт новин, клацаєте статтю — і сторінка раптом російською. Шукаєте в Google українське слово, а перші результати не тією мовою. Movar усуває обидва випадки — одна обрана мова застосовується у пошуку, на багатомовних сайтах і в перемикачах мови на сторінках. Це працює не лише для української: будь-який багатомовний користувач задає свою мову один раз, і Movar тримає її за вас.

Що це робить

- Переписує URL пошукових систем, щоб результати поверталися вашою мовою.
- Виявляє, коли сайт відкрився не тією мовою, і дає перемкнути її одним кліком.
- Приховує непотрібні мови у перемикачах на самих сайтах, щоб ті, які ви не використовуєте, не плуталися під ногами.

Підтримувані пошукові системи

Google (.com, .com.ua, .de, .fr, .co.uk, .pl, .com.au), Bing, DuckDuckGo, YouTube.

Доступні мови

Українська, англійська, німецька, французька, іспанська, італійська, польська.

Як це працює

- Один раз оберіть бажану мову на сторінці налаштувань.
- Movar застосовує виправлення під час перегляду.
- Відкрийте спливне вікно, щоб побачити лічильник виправлень за день, призупинити Movar для поточного сайту або перейти до налаштувань.

Приватність

- Без облікового запису, без входу.
- Без аналітики, без телеметрії.
- Нічого не покидає ваш пристрій. Налаштування зберігаються у браузері; визначення мови та переписування URL виконуються повністю локально.

Відкритий код

Movar — відкритий код під ліцензією MIT.

Скоро

Перемикання мови за пріоритетом — ви задаєте ранжований список мов, а Movar обирає найвищу доступну для кожного сайту.
